# FOCUS — Task Manager

A simple to-do web app for adding and managing daily tasks with priority labels.

## Main features
- Add a new task with a text field
- Set task priority: LOW, MED, HIGH
- Filter tasks by All, Active, Done, High, Med, Low
- Show total, done, and remaining task counts
- Save tasks in browser storage so they stay after refresh

## Files
- `myapp.html` - app structure and content
- `style.css` - app styling and layout
- `index.js` - task logic and storage handling

## How to use
1. Open `myapp.html` in your browser.
2. Type a task and click `ADD` or press Enter.
3. Choose a priority and use filters to view tasks.
4. Click `CLEAR DONE` to remove completed tasks.
